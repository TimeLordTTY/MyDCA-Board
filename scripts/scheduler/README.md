# 定时任务调度模块

## 功能说明

使用APScheduler实现定时数据采集和指标计算任务。

## 定时任务配置

1. **基金净值采集**：每天 18:00
2. **ETF实时行情**：交易时间内每5分钟（9:30-15:00）
3. **ETF日K线**：每天 15:30（收盘后）
4. **指标计算**：每天 16:00（数据采集完成后）

## 使用方法

```bash
# 安装依赖
pip install -r requirements.txt

# 启动调度器
python scheduler.py
```

## 注意事项

- 调度器会持续运行，使用Ctrl+C停止
- 建议使用systemd或supervisor管理进程
- 生产环境建议配置日志记录
