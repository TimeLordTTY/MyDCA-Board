账户说明

账户结构：
• 平台（ROOT）：用于分组，如"华宝证券"、"支付宝"
• 信封（LEAF）：实际存储余额的账户，如"生活费"、"理财金"
资金用途（fund_usage）：
• SPENDABLE：可支出，用于日常消费
• RESERVED：专款，用于特定用途（如逆回购占用）
• INVESTABLE：可投资，用于购买理财产品
操作提示：
• 点击账户可以编辑余额、用途、状态
• 点击平台可以重命名
• 所有变更会实时反映在看板上

快速说明

这个 Demo 不做真实税务/批次成本，只做"看得见的联动"。
你的正式系统里是复式记账 + postings，这里用简化引擎模拟出"你点一下，数字就动"。
目的：把 UI/交互体验先做顺，再接后端。

你可以试试：

1）去【设置-账户】入金 → 分配到【理财金】
2）去【订单&结算】下一个 BUY → 确认成交
3）回到【总览】看净资产/配比/建议变化

结算说明（演示规则）

- BUY：PENDING 时只占用资金；CONFIRM 时扣余额 + 增加持仓
- SELL：CONFIRM 时减少持仓 + 增加余额（演示版不做复杂成本）
- 逆回购（BOND_REPO）：PENDING 占用本金；CONFIRM 时释放占用 + 记入利息收益（不生成持仓）

防呆（演示版）

- 支出只能从 SPENDABLE 的信封扣款
- ETF/基金 BUY 默认只能从 INVESTABLE 扣款
- 逆回购允许华宝平台任意信封来源（你设定的规则）
